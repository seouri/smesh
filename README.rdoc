= SMeSH: Suggests MeSH Headings 

== Live Demo

http://smesh.medvane.org

== Install and use
  $ [sudo] gem install bio -v 1.3.0
  $ [sudo] gem install sinatra
  $ git clone git://github.com/seouri/smesh.git
  $ cd smesh
  $ ruby suggest.rb

open http://localhost:4567 in a browser
