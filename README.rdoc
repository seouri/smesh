= MeSH Suggestions

== Install and use
  $ [sudo] gem install bio
  $ [sudo] gem install sinatra
  $ git clone git://github.com/seouri/mesh-suggestions.git
  $ cd mesh-suggestions
  $ ruby suggest.rb

open http://localhost:4567 in a browser
